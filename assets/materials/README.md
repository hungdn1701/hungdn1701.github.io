# Teaching Materials

This folder contains course materials organized by course and language.

## Structure

```
materials/
├── service-oriented-development-vi/   # Vietnamese version
│   ├── slides/          # Lecture slides (PDF)
│   ├── labs/            # Lab instructions
│   └── resources/       # Additional resources
└── service-oriented-development-en/   # English version
    ├── slides/
    ├── labs/
    └── resources/
```

## Naming Convention

- Slides: `week-01-introduction.pdf`, `week-02-api-design.pdf`
- Labs: `lab-01-rest-api.pdf`, `lab-02-docker.pdf`

## Usage in Course Pages

Link to materials using:
```markdown
[Slides Week 1](/assets/materials/service-oriented-development-vi/slides/week-01.pdf)
```
